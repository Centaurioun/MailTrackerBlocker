MailTrackerBlocker

A tracker and spy pixel blocker plugin for macOS Apple Mail.

Download v0.2.1 for macOS 10.15 View on GitHub

What is this?

I've been using the default Mail client built in macOS for a long time now and it works perfectly fine when I'm both at work and at home. My coworker one day sent me an email with a particuliar signature that claims to notify the sender when opened. Intrigued, I read into it a bit more and discovered there are services that allow users to embed a small image in emails (endearingly called "spy pixels" by some) to allow the sender to check if you read their email.

Hm. Coworkers checking up on me. That sounds lovely.

Luckily, The Verge wrote a piece on how prevent this. Disabling "load remote content in messages" works like a treat but you're stuck without seeing legitimate images. Better yet, there are even Chrome extensions that automatically block these spy pixels images but nothing for Apple Mail so hence this plugin.

Most commercial trackers can be reliably blocked through a blacklist of known URL patterns. Failing that, MailTrackerBlocker applies a generic regex filter for 1x1 images. Read more about how email pixel tracking works.

How do I use this?

To enable, first install the .pkg file then enable the plugin by (see similar guide for nice pictures):

  1. In Mail, open Preferences > General > Manage Plug-ins...
  2. Check MailTrackerBlocker.mailbundle
  3. Click Apply and Restart Mail
